# Avatar Wardrobe

[For the English version, click here](./README.md)

## インストール

Avatar Wardrobeを使用するには、以下のウェブサイトにアクセスしてください：

[Avatar Wardrobe](https://avatarwardrobe.vercel.app/)

## 機能

- お気に入りのアバターを保存
- 各アバターの新しい衣装を追跡
- JP/ENの言語サポート
- データのインポート/エクスポートにより、デバイス間で設定を移行可能
